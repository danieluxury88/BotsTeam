## Design Notes

### Visual Hierarchy
1. **Primary actions** - Large, high-contrast buttons
2. **Secondary actions** - Smaller, lower-contrast buttons
3. **Tertiary actions** - Icon buttons or text links

### Color Coding
- âœ… Green - Success, completed, active

### Spacing Guidelines
- **Mobile**: 16px base spacing, 24px between sections
- **Tablet**: 20px base spacing, 32px between sections
- **Desktop**: 24px base spacing, 48px between sections

### Typography Scale
- **Headlines**: 24-32px (mobile) to 32-48px (desktop)
- **Subheadings**: 18-20px (mobile) to 20-24px (desktop)
- **Body**: 16px (mobile) to 16-18px (desktop)
- **Small text**: 14px (mobile) to 14-16px (desktop)

### Card Sizes
- **Mobile**: Full width, stacked vertically
- **Tablet**: 2 columns, 48% width each
- **Desktop**: 3-4 columns, 30-24% width each
- **Wide screens**: 4-6 columns, 23-16% width each

## Implementation Notes

These wireframes should be implemented using:
- CSS Grid for main layout
- Flexbox for component internals
- CSS Custom Properties (variables) for consistent theming
- Media queries for responsive breakpoints
- Touch-friendly sizing (48px minimum touch targets)
- Semantic HTML5 elements for accessibility
