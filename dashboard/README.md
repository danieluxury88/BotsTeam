## Quick Start

### 1. Generate Dashboard Data

```bash
# From repository root
cd dashboard
python3 generate_data.py
```

This scans your `data/` directory and `~/.devbot/projects.json` to create dashboard data files.

### 2. Start the Server

```bash
python3 server.py
```

The dashboard will be available at: http://localhost:8080

### 3. Access the Dashboard

Open your browser to:
- **Main Dashboard**: http://localhost:8080/
- **Projects**: http://localhost:8080/projects.html
- **Bots**: http://localhost:8080/bots.html
- **Activity**: http://localhost:8080/activity.html

## Usage

### Generating Data

The dashboard displays data from JSON files generated by `generate_data.py`:

```bash
python3 generate_data.py
```

This creates three JSON files in `dashboard/data/`:
- `dashboard.json` - Summary statistics
- `projects.json` - Project information
- `index.json` - Reports index

### Refreshing Data

Run the data generator again anytime to update the dashboard:

```bash
cd dashboard
python3 generate_data.py
```

Then refresh your browser to see the latest data.

### Custom Port

Start the server on a different port:

```bash
python3 server.py 3000
```

## Troubleshooting

### No Data Displayed

1. Ensure `generate_data.py` has been run
2. Check that JSON files exist in `dashboard/data/`
3. Verify projects are registered in `~/.devbot/projects.json`
4. Check browser console for errors

### Server Won't Start

1. Ensure Python 3.6+ is installed
2. Check if port 8080 is already in use
3. Try a different port: `python3 server.py 3000`

### Reports Not Loading

1. Verify reports exist in `data/{project}/reports/`
2. Re-run `generate_data.py`
3. Check file permissions
